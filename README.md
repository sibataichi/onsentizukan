# <温泉地図鑑>

## サイト概要
日々の疲れを癒すことができる温泉地をシェアするサイト

### サイトテーマ
温泉地にある、食・文化・温泉・自然・芸術などを投稿したりできるSNSサイト

### テーマを選んだ理由
食・文化・温泉・自然・芸術を楽しめ日々の疲れを癒すことができ、リピーターも多い
温泉地を共有するコンテンツがあれば良いと思い、このテーマにしました。

### ターゲットユーザ
温泉が好きな人
旅行が好きな人

### 主な利用シーン
温泉地で人気のスポットやリピーターが知る穴場スポットを、投稿・検索したりできる
